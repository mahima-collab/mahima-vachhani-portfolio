# Mahima Vachhani — Portfolio

DevOps Engineer portfolio site built with React, TypeScript, Vite, and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

## GitHub Pages deploy

Push to `main` — the workflow builds with `GITHUB_PAGES=true` and deploys to Pages.

Repository name: `mahima-vachhani-portfolio`  
Live URL: `https://<github-username>.github.io/mahima-vachhani-portfolio/`
